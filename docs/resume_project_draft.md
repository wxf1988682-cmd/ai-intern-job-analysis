## 项目经历

### 基于 Python 的 AI 实习岗位信息爬取与数据分析系统

项目简介：
本项目面向 AI 实习、Python 开发、数据处理、AIGC 等方向岗位，使用 Python 对岗位信息进行采集、清洗、分析和可视化，统计岗位中的高频技能要求，并总结适合本科生准备实习的学习重点。

技术栈：
Python、Pandas、Numpy、requests、BeautifulSoup、Matplotlib、Seaborn、Git、GitHub

项目内容：

* 设计项目目录结构，按照数据采集、数据清洗、数据分析、可视化展示和项目文档进行模块划分；
* 计划采集岗位名称、公司、城市、薪资、学历要求、岗位描述、技能关键词等字段；
* 使用 Pandas 对岗位数据进行去重、缺失值处理和字段规范化；
* 统计 Python、Pandas、Numpy、爬虫、MySQL、机器学习、PyTorch、AIGC 等技能关键词出现频率；
* 使用 Matplotlib / Seaborn 绘制技能词频图、城市分布图和薪资分布图；
* 根据岗位数据分析结果，总结 AI / Python / 数据处理方向实习岗位的核心能力要求。

项目收获：

* 熟悉了 Python 数据分析项目的基本流程；
* 初步掌握了 GitHub 项目管理和 README 文档编写；
* 对 AI / Python / 数据处理方向实习岗位的技能要求有了更清晰的认识；
* 为后续简历投递和面试准备积累了可展示项目。
